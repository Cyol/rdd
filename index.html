Inspiration :

"Les Terres Médianes du Rêve" est une aide de jeu pour le jeu de rôle Rêve de Dragon.

Les outils de cette aide de jeu sont basés sur les règles de la 2ème édition de Rêve de Dragon.

Les Rencontres Alternatives dans les TMR sont tirée de l'aide de jeu "Faire du Vieux avec du Neuf" du Tinkle Bavard n°8, récupéré sur L'Omnirêve.

Développement :

HTML, CSS, JS : Cyol ( http://cyol.fr/blog | @CyolFalcon sur Twitter )

Autres :

  • Ce logiciel utilise les bibliothèques : jQuery 1.2.1, jQuery UI 1.11.4, jQuery mobile 1.4.5, jQuery UI Touch Punch 0.2.3
  • La génération de la carte des TMR (à venir) est une adaptation de la bibliothèque Hexagon.js
  • Le fond parcheminé des TMR vient de https://pixabay.com/
  • Les illustrations des TMRs viennent de la 2ème édition de Rêve de Dragon
